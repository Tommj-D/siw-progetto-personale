Caso d’uso UC1 – Visualizzazione del catalogo dei libri
Un Cliente utilizza il Sistema per visualizzare il catalogo dei libri disponibili.
Il Sistema mostra l’elenco completo dei libri presenti nel catalogo.

Caso d’uso UC2 – Visualizzazione dei dettagli di un libro
Un Cliente utilizza il Sistema per visualizzare i dettagli di un libro.
Il Sistema mostra le informazioni del libro selezionato (titolo, autore, prezzo, descrizione).

Caso d’uso UC3 – Ricerca di libri per titolo o autore
Un Cliente utilizza il Sistema per cercare libri in base al titolo o all’autore.
Il Sistema mostra l’elenco dei libri che corrispondono ai criteri di ricerca.

Caso d’uso UC4 – Inserimento di un nuovo libro
Un Admin utilizza il Sistema per inserire un nuovo libro nel catalogo.
Il Sistema salva il nuovo libro nel catalogo.

Caso d’uso UC5 – Modifica di un libro esistente
Un Admin utilizza il Sistema per modificare i dati di un libro esistente.
Il Sistema aggiorna le informazioni del libro (ad esempio prezzo e descrizione).

Caso d’uso UC6 – Cancellazione di un libro
Un Admin utilizza il Sistema per eliminare un libro dal catalogo.
Il Sistema rimuove il libro dal catalogo.

Caso d’uso UC7 – Aggiunta di un libro al carrello
Un Cliente utilizza il Sistema per aggiungere un libro al carrello.
Il Sistema inserisce il libro selezionato nel carrello del Cliente.

Caso d’uso UC8 – Effettuare un ordine
Un Cliente utilizza il Sistema per effettuare un ordine dei libri presenti nel carrello.
Il Sistema registra l’ordine e associa i libri acquistati al Cliente.

Caso d’uso UC9 – Visualizzazione degli ordini
Un Cliente utilizza il Sistema per visualizzare i propri ordini.
Il Sistema mostra l’elenco degli ordini effettuati dal Cliente.
